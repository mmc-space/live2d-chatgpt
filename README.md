[这里]()去下载最新版本，Mac 使用.dmg， Window 使用.exe，Linux 使用.deb。

## 使用

### 导入模型

两种方式：

- 在设置 -> 模型列表中添加 `json` 文件（本地模型以`file://`开头）。

- 本地模型将文件夹拖入窗口中，导入成功会更新到模型列表中
